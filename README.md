# Responsive Admin Dashboard

## Project Overview

This is a modern, responsive admin dashboard UI built as part of a frontend development assignment. The dashboard is designed to work seamlessly across desktop, tablet, and mobile devices, demonstrating proficiency in HTML5, CSS3, and vanilla JavaScript.

## Tech Stack

- **HTML5**: Semantic markup and modern HTML features
- **CSS3**: Responsive design, animations, and modern styling
- **Tailwind CSS**: Utility-first CSS framework for rapid development
- **JavaScript (Vanilla)**: Interactive functionality and DOM manipulation
- **Font Awesome**: Icon library for UI elements
- **Chart.js**: Data visualization library for charts

## Features

- **Fully Responsive Design**: Optimized for desktop (1200px+), tablet (768px-1199px), and mobile (320px-767px)
- **Clean Code Structure**: Well-organized, maintainable code with no inline CSS
- **Modern UI/UX**: Professional dashboard layout with intuitive navigation
- **Cross-browser Compatibility**: Works on all modern browsers

## How to Run the Project

### Method 1: Direct File Opening
1. Download or clone the project folder
2. Simply open `index.html` in your web browser

### Method 2: Local Server (Recommended)
1. Download or clone the project folder
2. Navigate to the project directory in your terminal
3. Start a local server:
   - Using Python: `python -m http.server 8000`
   - Using Node.js: `npx serve`
   - Using Live Server extension in VS Code
4. Open `http://localhost:8000` in your browser
